Welcome to The Gesture Controlled Turtlebot 4 Documentation!
===================================

We use ROS2 Galactic to perform the gesture control for real time collaborative control of Turtlebot 4.

Check out the :doc:`usage` section for further information, including
how to :doc:`getting started` with the project. If you have everything setup and want a quick start, jump to https://hand-gesture-controlled-turtlebot.readthedocs.io/en/latest/usage.html#quick-start

.. note::

   This project is under active development.
